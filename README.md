## Configuração do Ambiente

1. Clone o repositório:  
   ```bash
   git clone https://github.com/gabrielabade/devburger-interface
   ```  
2. Acesse a pasta do projeto:  
   ```bash
   cd devburger-interface
   ```  
3. Instale as dependências:  
   ```bash
   yarn install
   ```  
4. Execute o projeto:  
   ```bash
   yarn dev
   ```
